# VXT Korea WebPage
베트남 맨파워 회사인 VXT 의 한국 지부 설립을 위해 제작한 홈페이지 프로젝트 입니다.

## 💻 프로젝트 소개
React.js 와 Express.js 로 제작한 VXT KOREA 의 홈페이지 입니다.

## ⏲️ 개발 기간
2023.12 - 2024.01

### ⚙️ 개발 환경
<ul>
  <li>React.js, Node.js, TypeScript</li>
  <li>IDE : VScode</li>
  <li>Deploy: AWS S3, EC2, Cloud Front, Route53, Cafe24</li>
</ul>
